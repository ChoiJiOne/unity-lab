# Kenney Racing Pack 에셋 가이드

탑다운 레이싱 게임용 2D 에셋 팩입니다.
자동차·오토바이·트랙 타일·장애물로 위에서 본 레이싱을 구성할 수 있습니다.

- **경로:** `Assets/Arts/kenney_racing-pack/`
- **제작자:** Kenney (www.kenney.nl) · Racing Pack
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

위에서 본 **레이싱용 차량 + 트랙 타일 세트**입니다. 플랫/벡터 스타일.
PNG 429개 + SVG/SWF 벡터 + 스프라이트시트로 구성됩니다.

| 카테고리 | 개수 | 내용 |
|----------|------:|------|
| **Cars** | 50 | 자동차 — 5색 × 일반/소형, 색상별 변형 |
| **Objects** | 39 | 콘, 타이어, 배리어, 결승선, 오일·웅덩이, 깃발 등 |
| **Characters** | 17 | 드라이버/관련 캐릭터 |
| **Motorcycles** | 5 | 오토바이 |
| **Tiles / Asphalt·Dirt·Sand road** | 각 90 | 노면별 트랙 타일(직선/코너/교차/체크) |
| **Tiles / Grass·Dirt·Sand** | 각 14 | 트랙 주변 지형 |
| **Spritesheets** | 4 | 통합 시트 + XML |

> **3가지 노면(아스팔트/흙/모래) × 코너·교차 트랙 타일** 풀세트 → 어떤 형태의 서킷도 타일로 구성. 차량은 위에서 본 5색.

---

## 2. 언제 사용하면 되는가?

- **탑다운 레이싱** — 위에서 본 서킷 레이스, 드리프트
- 트랙 에디터/서킷 빌더 (코너·교차 타일)
- 장애물 회피·체크포인트 레이스
- 멀티플레이어 색상 구분 차량(5색)

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **파일명이 의미 기반**(`car_blue_3`, `car_red_small_1`) → 색/크기로 식별·로딩 쉬움.
- **노면 3종 트랙 타일** — 아스팔트/흙/모래, 각 90개로 곡선 서킷 구성 가능.
- **차량 5색 × 소형 변형** — 플레이어/AI 구분.
- **Tilemap 적합** — 트랙 타일을 Tile Palette로 등록해 서킷 제작.
- **플랫 벡터 스타일** — 픽셀아트 아님.

---

## 형태 선택 가이드

- **플레이어/AI 차량:** `PNG/Cars/`, `PNG/Motorcycles/`
- **서킷 트랙:** `PNG/Tiles/<노면> road/`(Tilemap)
- **장애물/연출:** `PNG/Objects/`
- **시트 일괄 임포트:** `Spritesheets/` + XML
- **고해상도/벡터:** `Vector/` (SVG)
- **참고:** `Preview.png`(전체), `Sample.png`(배치 예시)

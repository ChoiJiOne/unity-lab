# Kenney Generic Items 에셋 가이드

일상 사물/도구 아이템 아이콘 팩입니다.
공구·전자기기·주방용품·문구·과학장비 등 범용 아이템을 인벤토리/UI에 표시할 때 사용합니다.

- **경로:** `Assets/Arts/kenney_generic-items/`
- **제작자:** Kenney (www.kenney.nl) · Generic Items #1
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

범용 일상 사물 **아이템 아이콘 세트**입니다. 플랫/벡터 스타일.
163종 × (Colored / White) = PNG 326개 + SVG 벡터 + 스프라이트시트로 구성됩니다.

| 구분 | 개수 | 내용 |
|------|------:|------|
| **PNG / Colored** | 163 | 채색 버전 아이템 |
| **PNG / White** | 163 | 흰색 단색 실루엣(틴팅/아이콘용) |
| **Spritesheet** | 2 | 통합 시트 + XML |
| **Vector** | SVG/SWF | 무한 확대용 원본 |

**소재:** 공구(망치/톱/렌치/드릴), 전자기기(노트북/모니터/폰/게임패드), 주방용품(냄비/팬/컵/포크), 문구(책/클립보드/펜), 과학장비(현미경/플라스크/비커), 가방·열쇠 등 생활 사물.

> 같은 163종이 **Colored(채색)** 와 **White(단색)** 두 버전 → 채색은 아이템 그래픽, White는 틴팅 가능한 UI 아이콘.

---

## 2. 언제 사용하면 되는가?

- **인벤토리 / 아이템 아이콘** — 수집·제작(crafting) 게임
- **상점 UI / 퀵슬롯**
- 시뮬레이션·생활 게임의 도구·가전 표현
- White 버전을 **틴팅**해 등급·상태별 아이콘 색 구분
- 범용 UI 아이콘(공구·과학·주방 카테고리)

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **Colored / White 2버전** — White는 흰색 실루엣이라 color 틴팅에 적합.
- **파일명이 `genericItem_color_001` / `_white_001` 식 인덱스** — 어떤 아이템인지는 `Preview.png`로 매칭 필요.
- **투명 배경 PNG** — 인벤토리 슬롯 위 오버레이 용이.
- **플랫 벡터 스타일** — 픽셀아트 아님.
- 시리즈명이 "#1" — 동일 톤의 다른 아이템 팩과 조합 가능.

---

## 형태 선택 가이드

- **아이템 그래픽:** `PNG/Colored/`
- **틴팅 가능한 UI 아이콘:** `PNG/White/`
- **시트 일괄 임포트:** `Spritesheet/` + XML
- **고해상도/벡터:** `Vector/` (SVG)
- **참고:** `Preview.png`(전체 카탈로그)

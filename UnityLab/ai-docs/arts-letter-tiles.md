# Kenney Letter Tiles 에셋 가이드

알파벳 글자 타일 팩입니다.
스크래블/단어 맞추기 게임의 문자 타일을 표시할 때 사용합니다.

- **경로:** `Assets/Arts/kenney_letter-tiles/`
- **제작자:** Kenney (www.kenney.nl) · Letter Tiles
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

단어 게임용 **알파벳 타일 세트**입니다. 플랫/벡터 스타일.
A~Z + 빈 타일 × 7가지 재질/색 = PNG 226개 + SVG/AI 벡터 + 스프라이트시트로 구성됩니다.

| 재질/색(폴더) | 내용 |
|----------------|------|
| **Blue / Yellow** | 컬러 타일 |
| **Brown / Wood** | 목재 톤 (스크래블 클래식) |
| **Marble / Metal** | 대리석 / 금속 질감 |
| **Box / Solid** | 박스형 / 단색 |

각 폴더: `letter_A`~`letter_Z`(26) + `letter`(빈 타일) = 27개. **Spritesheet** 7장(재질별) + XML.

> 스크래블처럼 **글자 + 점수 숫자**가 타일에 함께 그려진 디자인. 7가지 재질로 테마 선택.

---

## 2. 언제 사용하면 되는가?

- **단어 맞추기 게임** — 스크래블, 워들류, 행맨, 단어 조합 퍼즐
- **타일 기반 철자/교육** 게임
- 알파벳 학습 UI, 이름 입력 연출
- 글자 블록이 필요한 캐주얼 퍼즐

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **글자 타일에 점수 숫자 포함**(스크래블 스타일) — 단어게임 점수 룰에 바로 사용.
- **7가지 재질/색** — 게임 테마(목재/금속/컬러)에 맞춰 선택.
- **파일명이 의미 기반**(`letter_A`, `letter`(빈칸)) → 글자 단위 로딩 매우 쉬움.
- **빈 타일(`letter.png`)** 포함 — 와일드카드/공백 표현.
- **벡터(SVG/AI) 포함** — 고해상도/인쇄.
- **플랫 벡터 스타일** — 픽셀아트 아님.

---

## 형태 선택 가이드

- **글자 타일:** `PNG/<재질>/letter_X`
- **빈/와일드카드:** `PNG/<재질>/letter.png`
- **시트 일괄 임포트:** `Spritesheet/` + XML
- **고해상도/벡터:** `Vector/` (SVG/AI)
- **참고:** `Preview.png`(재질 비교), `Sample.png`(배치 예시)

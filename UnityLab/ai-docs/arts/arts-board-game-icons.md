# Kenney Board Game Icons 에셋 가이드

보드게임 규칙/액션을 나타내는 단색 아이콘 세트입니다.
카드·주사위·이동·자원 등 게임 행동을 픽토그램으로 표기할 때 사용합니다.

- **경로:** `Assets/Arts/kenney_board-game-icons/`
- **제작자:** Kenney (www.kenney.nl) · Board Game Icons v1.1 (2024-07-22)
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

보드게임 룰북/UI용 **흰색 단색 픽토그램 아이콘 세트**입니다. (게임 그래픽이 아니라 "기호")
255종 아이콘 × 2해상도 + 벡터로 구성됩니다.

| 형식 | 개수 | 내용 |
|------|------:|------|
| **PNG / Default (64px)** | 255 | 기본 해상도 아이콘 |
| **PNG / Double (128px)** | 255 | 2배 해상도 |
| **Vector / Icons (SVG)** | 256 | 무한 확대용 벡터 (개별 SVG) |
| **Tilesheet** | — | 통합 시트 + 매핑 |

**아이콘 주제(예시):** 카드 액션(card_tap, card_flip, card_place, card_draw…), 주사위(D4·D6·D8·D10·D12·D20), 말/폰/기물(체스 기물·하우스·타워), 이동·회전 화살표(arrow_rotate, arrow_clockwise…), 자원·점수($, award, heart), 방향(N·E·S·W), 책·캠프파이어·불꽃 등 테마 심볼.

> 흰색 실루엣 단일 색 → **틴팅(tint)으로 원하는 색 적용** 가능. 게임 룰 설명·액션 버튼에 최적.

---

## 2. 언제 사용하면 되는가?

- **보드/카드 게임 UI** — 액션 버튼, 턴 표시, 규칙 안내 아이콘
- **룰북/툴팁** 픽토그램 (카드 뽑기/내기/뒤집기, 주사위 굴림 등)
- 주사위 종류(D4~D20) 표기, 자원·점수 심볼
- 보드게임 외에도 범용 **UI 아이콘**으로 재사용 (화살표·회전·방향)
- [arts-boardgame-pack.md](arts-boardgame-pack.md)(실제 카드/말/주사위 그래픽)와 **짝**으로 사용하면 좋음.

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **단색 흰색 아이콘** — Unity `Image`/`SpriteRenderer`의 color로 틴팅해 테마 색 적용.
- **파일명이 의미 기반**(`card_flip`, `arrow_clockwise`, `dice_d20`) → 행동 단위로 직접 찾기 쉬움. (이 팩은 인덱스명이 아님)
- **64px / 128px 2해상도 + SVG** — UI 크기·DPI에 맞춰 선택.
- **투명 배경 PNG** — 어떤 배경 위에도 오버레이 가능.
- 게임플레이 스프라이트가 아닌 **기호/UI 아이콘**이라는 점에 유의.

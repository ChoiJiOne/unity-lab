# Kenney Tower Defense (Top-Down) 에셋 가이드

탑다운 타워디펜스 게임용 2D 에셋 팩입니다.
경로 타일·타워·적·발사체와 숫자 HUD로 TD를 구성할 수 있습니다.

- **경로:** `Assets/Arts/kenney_tower-defense-top-down/`
- **제작자:** Kenney (www.kenney.nl) · Tower Defense (top-down) Pack
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

위에서 본 **타워디펜스용 타일 + 타워/적/발사체 세트**입니다. 플랫/벡터 스타일.
PNG 603개(Default size + Retina 2배) + SVG/SWF 벡터 + 타일시트로 구성됩니다.

| 카테고리 | 내용 |
|----------|------|
| **Path / Terrain 타일** | 적 이동 경로(흙길), 잔디·돌·물 배치 타일, 코너·교차 경로 |
| **Towers (타워)** | 포탑 베이스 + 분리형 포신(여러 등급), 미사일/캐넌 타입 |
| **Enemies** | 비행기·차량 등 진격 유닛 |
| **Projectiles / Effects** | 총알·미사일·폭발·화염 이펙트 |
| **HUD** | 숫자 0~9, `$`(자원), 하트/아이콘 등 상태 표시 |
| **Default size / Retina** | 동일 세트의 1배 / 2배 해상도 |
| **Tilesheet** | 통합 타일시트 |

> **경로(path) 타일이 코너/교차 포함 풀세트** → 적 이동 경로를 타일로 깔고 웨이포인트만 지정하면 됨. 타워는 베이스+포신 분리.

---

## 2. 언제 사용하면 되는가?

- **타워디펜스(TD)** — 경로 방어, 타워 설치/업그레이드
- 웨이포인트 기반 적 이동 + 포탑 자동 조준
- 자원($)·체력(하트) HUD 표시
- 격자 기반 배치 전략 게임 프로토타입

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **파일명이 `towerDefense_tile001` 식 인덱스** — 어떤 타일/타워인지는 `Preview.png` / `Tilesheet`로 매칭 필요.
- **경로 타일 풀세트(코너/교차)** — 곡선·분기 경로를 타일만으로 구성.
- **타워 베이스+포신 분리** — 포신만 회전시켜 적 조준.
- **Default / Retina 2단계 해상도** — 고DPI는 Retina.
- **숫자/자원 HUD 포함** — 점수·골드·웨이브 표시에 활용.
- **플랫 벡터 스타일** — 픽셀아트 아님.

## 형태 선택 가이드

- **맵/경로:** `Tilesheet/`(Tilemap) 또는 `towerDefense_tile*` 낱개
- **타워:** 베이스 + 포신 분리 조합(포신 회전)
- **적/발사체/이펙트:** 해당 인덱스 스프라이트
- **HUD:** 숫자·`$`·아이콘 스프라이트
- **고해상도:** `PNG/Retina/`
- **참고:** `Preview.png`(전체), `Sample.png`(배치 예시)

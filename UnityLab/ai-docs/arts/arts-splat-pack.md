# Kenney Splat Pack 에셋 가이드

물감/잉크 튀김(splat) 모양 텍스처 팩입니다.
얼룩·페인트 자국·데칼·마스크 등 다양한 splat 효과에 사용합니다.

- **경로:** `Assets/Arts/kenney_splat-pack/`
- **제작자:** Kenney (www.kenney.nl) · Splat Pack v1.0 (2022-03-16)
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

물감 튀김 형태의 **흰색 실루엣 splat 텍스처 세트**입니다.
36종 × (256px / 512px) = PNG 72개 + SVG 벡터 37개로 구성됩니다.

| 구분 | 개수 | 내용 |
|------|------:|------|
| **PNG / Default (256px)** | 36 | 기본 해상도 splat 실루엣 |
| **PNG / Double (512px)** | 36 | 2배 해상도 |
| **Vector** | 37 | SVG 원본 (무한 확대) |

**형태:** 둥근 얼룩, 튀는 물방울, 길게 번진 자국, 작은 점 흩뿌림 등 36가지 splat 모양.

> **흰색 단색 실루엣** → color 틴팅으로 페인트/피/슬라임/물 등 어떤 색이든 표현. 데칼·마스크로 광범위 활용.

---

## 2. 언제 사용하면 되는가?

- **스플래툰류 페인트** — 바닥/벽 물감 데칼
- **피격/타격 효과** — 피·슬라임·진흙 튀김
- 트랜지션 마스크, 얼룩 오버레이
- 파티클·데칼 시스템의 모양 텍스처
- UI 장식(거친 배경 얼룩, 강조 배경)

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **흰색 단색 실루엣** — Unity color/머티리얼로 틴팅해 원하는 색(재사용 효율적).
- **256px / 512px + SVG** — 용도/해상도에 맞춰 선택. 큰 데칼은 512px 또는 SVG.
- **파일명이 `splat00` 식 인덱스** — 형태는 `Preview.png`로 매칭.
- **투명 배경 PNG** — 데칼/오버레이 합성 용이.
- 데칼 프로젝터나 스프라이트 마스크와 조합하면 효과적.

---

## 형태 선택 가이드

- **일반 데칼/얼룩:** `PNG/Default (256px)/`
- **큰 화면/근접 데칼:** `PNG/Double (512px)/`
- **확대·축소 잦은 마스크:** `Vector/` (SVG)
- **참고:** `Preview.png`(전체 형태)

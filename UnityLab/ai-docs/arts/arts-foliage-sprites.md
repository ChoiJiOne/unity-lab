# Kenney Foliage Sprites 에셋 가이드

식물/초목 실루엣 스프라이트 팩입니다.
잔디·풀·잎·꽃 등으로 자연 배경을 장식하거나 빌보드 식생을 깔 때 사용합니다.

- **경로:** `Assets/Arts/kenney_foliage-sprites/`
- **제작자:** Kenney (www.kenney.nl) · Foliage Sprites (2020-02-08)
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

자연 배경용 **식물/초목 스프라이트 세트**입니다.
50종 × (Flat / Shaded) = PNG 100개 + SVG 벡터로 구성됩니다.

| 구분 | 개수 | 내용 |
|------|------:|------|
| **PNG / Flat** | 50 | 단색(흰색) 실루엣 — 틴팅 용이 |
| **PNG / Shaded** | 50 | 음영이 들어간 버전 |
| **Vector** | 2 | SVG 원본 |

**소재:** 잔디 다발, 키 큰 풀, 갈대, 단풍/넓은 잎, 클로버, 꽃 줄기, 덤불, 민들레 홀씨 등 다양한 식생 실루엣.

> 같은 50종이 **Flat(단색)** 과 **Shaded(음영)** 두 버전으로 제공 → Flat은 색을 입혀 분위기 통일, Shaded는 그대로 디테일 표현.

---

## 2. 언제 사용하면 되는가?

- **2D/3D 자연 배경 장식** — 잔디·풀숲·잎 빌보드
- 탑다운/사이드뷰 맵의 **식생 디테일** 추가
- 단색(Flat) 실루엣을 **틴팅**해 계절·바이옴 색감 연출
- 파티클/바람 흔들림용 잎 소재

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **Flat은 흰색 실루엣** — Unity color로 틴팅해 원하는 식물 색 적용(여러 색 재사용에 효율적).
- **파일명이 `sprite_0001` 식 인덱스** — 어떤 식물인지는 `Preview.png`로 매칭 필요.
- **Flat / Shaded 선택** — 톤에 따라. Flat=색 자유, Shaded=즉시 디테일.
- 투명 배경 PNG → 어떤 배경 위에도 오버레이.
- 바람에 흔들리는 풀 셰이더(예: sprite sway)와 조합하면 효과적.

---

## 형태 선택 가이드

- **색을 직접 입힐 때:** `PNG/Flat/`(틴팅)
- **음영 포함 즉시 사용:** `PNG/Shaded/`
- **벡터:** `Vector/` (SVG)
- **참고:** `Preview.png`(전체), `Sample.png`(배치 예시)

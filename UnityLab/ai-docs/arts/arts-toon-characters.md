# Kenney Toon Characters 에셋 가이드

만화풍 2D 캐릭터 팩입니다.
6종 캐릭터의 다양한 포즈와 분리된 신체 파츠로 캐릭터를 구성·애니메이션할 수 있습니다.

- **경로:** `Assets/Arts/kenney_toon-characters/`
- **제작자:** Kenney (www.kenney.nl) · Toon Characters (2019-09-26)
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

귀여운 만화풍 **인간형 2D 캐릭터 세트**입니다. 플랫/벡터 스타일.
6종 캐릭터 × (완성 포즈 + 분리 파츠) = PNG 677개 + SVG 벡터 + 타일시트로 구성됩니다.

| 캐릭터(폴더) | 6종 |
|--------------|-----|
| Female adventurer, Female person, Male adventurer, Male person, Robot, Zombie |

각 캐릭터 폴더 구성:

| 하위 | 내용 |
|------|------|
| **Poses** | **포즈 45종** — walk, jump, attack, duck, cheer, hurt 등 완성 스프라이트 |
| **Poses HD** | 포즈 고해상도 버전 |
| **Parts** | 분리 신체 파츠 — head/hand/arm/leg 등 (본/리깅·종이인형 애니용) |
| **Parts HD** | 파츠 고해상도 |
| **Tilesheet / Vector** | 통합 시트 + SVG |

> 캐릭터당 **완성 포즈 45종**으로 즉시 사용하거나, **분리 파츠**로 본 리깅(2D 스켈레탈) 애니메이션 가능. 두 워크플로 모두 지원.

---

## 2. 언제 사용하면 되는가?

- **플랫포머/액션 주인공·NPC** — 6종(모험가/일반인/로봇/좀비)
- **포즈 스왑 애니메이션** — Poses 45종으로 상태 전환
- **본 리깅(2D 스켈레탈)** — Parts로 종이인형식 애니(Unity 2D Animation/스파인풍)
- 캐주얼/카툰 톤 게임의 캐릭터

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **두 워크플로 지원** — (1) **Poses** 완성 스프라이트 프레임 스왑, (2) **Parts** 분리 파츠 본 리깅.
- **HD/일반 2해상도** — Poses HD / Parts HD 별도.
- **파일명이 의미 기반**(포즈/파츠명) → 상태·부위 식별 쉬움.
- **6 캐릭터 동일 구조** — 캐릭터 교체가 폴더 단위로 일관됨.
- **플랫 벡터 스타일** — 픽셀아트 아님.

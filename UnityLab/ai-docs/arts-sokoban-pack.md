# Kenney Sokoban Pack 에셋 가이드

상자 밀기(소코반) 퍼즐 게임용 2D 에셋 팩입니다.
플레이어·상자(crate)·목표 블록·바닥 타일로 소코반 퍼즐을 구성할 수 있습니다.

- **경로:** `Assets/Arts/kenney_sokoban-pack/`
- **제작자:** Kenney (www.kenney.nl) · Sokoban Pack
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

격자 기반 **상자 밀기 퍼즐 세트**입니다. 플랫/벡터 스타일.
99종 × (Default size / Retina) = PNG 약 200개 + SVG 벡터 + 시트로 구성됩니다.

| 카테고리 | 개수 | 내용 |
|----------|------:|------|
| **Crates** | 45 | 상자 — 색상별, 일반/목표 위(놓임)/잠금 변형 |
| **Player** | 24 | 플레이어 캐릭터 — 4방향 × 이동 프레임 |
| **Environment** | 16 | 벽, 장식, 마커 |
| **Blocks** | 8 | 벽/블록 타일 |
| **Ground** | 6 | 바닥 타일(목표 지점 포함) |
| **Default size / Retina** | — | 1배 / 2배 해상도 |
| **Spritesheet / Tilesheet** | 4 | 통합 시트 + XML |

> **상자 색상 = 목표 지점 색상** 매칭 구조 → 같은 색 목표에 상자 밀어넣기 룰 구현. 플레이어 4방향 프레임 포함.

---

## 2. 언제 사용하면 되는가?

- **소코반(Sokoban)** — 상자를 목표로 미는 격자 퍼즐
- **격자 기반 논리 퍼즐** — 푸시/블록 이동
- 색 매칭 목표(상자→같은 색 패드) 퍼즐
- 퍼즐 게임 레벨 디자인 학습/프로토타입

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **상자/목표 색상 매칭** — 색별 crate와 대응 ground 마커로 룰 표현.
- **플레이어 4방향 + 프레임** — 격자 이동 애니메이션.
- **파일명이 의미 기반**(`crate_01`, `block_05`, `player_05`) → 식별 쉬움(범주는 폴더로 구분).
- **Default / Retina 2해상도** — 고DPI는 Retina.
- **Tilesheet 제공** — Tilemap으로 벽/바닥 구성.
- **플랫 벡터 스타일** — 픽셀아트 아님.

---

## 형태 선택 가이드

- **플레이어:** `Player/`(방향별 프레임)
- **밀 상자:** `Crates/`(색/상태별)
- **벽/바닥/목표:** `Blocks/`, `Ground/`, `Environment/`(Tilemap)
- **고해상도:** `PNG/Retina/`
- **시트 일괄 임포트:** `Spritesheet/`·`Tilesheet/` + XML
- **참고:** `Preview.png`(전체), `Sample.png`(배치 예시)

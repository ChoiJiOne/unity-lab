# Kenney Tappy Plane 에셋 가이드

탭 조작 사이드스크롤(Flappy Bird류) 게임용 2D 에셋 팩입니다.
비행기·지형·장애물과 비트맵 폰트로 원탭 아케이드를 빠르게 구성할 수 있습니다.

- **경로:** `Assets/Arts/kenney_tappy-plane/`
- **제작자:** Kenney (www.kenney.nl) · Tappy Plane v1.1
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

원탭 사이드스크롤(Flappy Bird 스타일) 게임용 **비행기 + 지형 + 장애물 + 비트맵 폰트 세트**입니다. 플랫/벡터 스타일.
PNG 83개 + SVG 벡터 + 스프라이트시트로 구성됩니다.

| 카테고리 | 개수 | 내용 |
|----------|------:|------|
| **Planes** | 12 | 비행기 — 4색(yellow/green/blue/red) × 프로펠러 애니메이션 프레임 |
| **Letters** | 26 | 비트맵 알파벳 A~Z |
| **Numbers** | 10 | 비트맵 숫자 0~9 (점수 표시용) |
| **UI** | 12 | TAP 버튼, 별/메달, GET READY·GAME OVER, 패널 |
| **PNG (루트)** | 19 | 지형(groundDirt/Grass/Ice/Rock/Snow), 배경, 바위/산 장애물(rock/spike) |
| **Spritesheet** | 2 | 통합 시트 + XML |

> 지형이 **5종(흙/잔디/얼음/바위/눈)**, 비행기 **4색 + 프로펠러 프레임** → 무한 스크롤 + 캐릭터 애니메이션 즉시 구현. 점수용 비트맵 폰트 포함.

---

## 2. 언제 사용하면 되는가?

- **원탭 아케이드** — Flappy Bird류 끝없이 나는 게임
- **사이드스크롤 러너 / 회피** 게임
- 무한 스크롤 지형 + 장애물(스파이크) 통과
- 점수·메달 UI가 필요한 하이퍼캐주얼 프로토타입

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유. (제작 2014년, v1.1)
- **파일명이 의미 기반**(`planeBlue1~3`, `groundGrass`, `letterA`, `rockGrass`) → 식별·애니메이션 묶기 쉬움.
- **비트맵 폰트(Letters/Numbers) 제공** — 폰트 없이 점수·문구 표시 가능.
- **프로펠러 애니메이션 프레임** — 비행기당 다중 프레임으로 회전 표현.
- **지형 타일링** — `ground*`는 가로 무한 스크롤에 적합.
- **플랫 벡터 스타일** — 픽셀아트 아님.

## 형태 선택 가이드

- **플레이어 비행기:** `PNG/Planes/`(색·프레임 선택)
- **무한 스크롤 지형/배경:** `PNG/groundXXX`, `PNG/background`
- **장애물:** `PNG/rock*` / 스파이크류
- **점수·UI:** `PNG/Numbers`, `PNG/Letters`, `PNG/UI`
- **시트 일괄 임포트:** `Spritesheet/` + XML
- **참고:** `Preview.png`(전체), `Sample.png`(배치 예시)

# Kenney Jumper Pack 에셋 가이드

수직 점프(Doodle Jump류) 캐주얼 게임용 2D 에셋 팩입니다.
점프 발판·아이템·적·배경으로 끝없이 올라가는 게임을 빠르게 구성할 수 있습니다.

- **경로:** `Assets/Arts/kenney_jumper-pack/`
- **제작자:** Kenney (www.kenney.nl) · Jumper Pack
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

위로 점프하며 올라가는 **버티컬 점퍼(Doodle Jump 스타일) 캐주얼 게임 에셋**입니다.
플랫/벡터 스타일, PNG 124개 + SVG/SWF 벡터 + 스프라이트시트로 구성됩니다.

| 카테고리 | 개수 | 내용 |
|----------|------:|------|
| **Players** | 12 | 토끼(bunny) 캐릭터 — 점프/이동/색상 변형 프레임 |
| **Enemies** | 22 | flyMan, spikeMan, springMan, wingMan, spikeBall, sun, cloud 등 |
| **Environment** | 35 | 점프 발판(플랫폼), 지형, 선인장·버섯 등 장식 |
| **Items** | 29 | 코인, 젬, 스프링, 부스터 등 수집·강화 아이템 |
| **Particles** | 14 | 점프/획득 이펙트 파티클 |
| **HUD** | 5 | 점수·UI 요소 |
| **Background** | 4 | `bg_layer1~4` — 패럴랙스 스크롤용 레이어 배경 |

> 배경이 **4겹 레이어(bg_layer)** 로 분리 → 패럴랙스 스크롤 즉시 구현 가능.

---

## 2. 언제 사용하면 되는가?

- **버티컬 점퍼** — Doodle Jump, Icy Tower류 끝없이 올라가는 게임
- **캐주얼/하이퍼캐주얼 모바일 게임** 프로토타입
- 점프 발판·스프링·부스터 기반 플랫폼 점프 메커닉
- 패럴랙스 배경 스크롤 테스트

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **파일명이 의미 기반**(`spikeMan_walk1`, `flyMan_fly`, `bg_layer1`) → 이름만으로 식별·애니메이션 프레임 묶기 쉬움.
- **적/플레이어 애니메이션 프레임 분리 제공** — `_walk1/_walk2`, `_stand/_jump` 식으로 상태별 스프라이트가 나뉨.
- **패럴랙스 배경 4레이어** 제공.
- **플랫 벡터 스타일** — 픽셀아트 아님. 임포트 시 일반 필터(Bilinear) 권장.
- **시트/낱개 선택** — `Spritesheets/`(1장 + XML) 또는 `PNG/` 낱개.

---

## 형태 선택 가이드

- **개별 스프라이트 제어:** `PNG/` 낱개
- **시트 일괄 임포트:** `Spritesheets/` + XML
- **고해상도/벡터:** `Vector/` (SVG)
- **참고:** `Preview.png`(전체), `Sample.png`(배치 예시)

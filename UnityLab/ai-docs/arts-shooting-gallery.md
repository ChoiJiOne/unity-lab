# Kenney Shooting Gallery 에셋 가이드

사격장(오리 쏘기) 게임용 2D 에셋 팩입니다.
과녁·오리·크로스헤어와 점수 HUD로 클래식 슈팅 갤러리를 구성할 수 있습니다.

- **경로:** `Assets/Arts/kenney_shooting-gallery/`
- **제작자:** Kenney (www.kenney.nl) · Shooting Gallery Pack
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

사격장 미니게임용 **표적 + 사격장 배경 + HUD 세트**입니다. 플랫/벡터 스타일.
PNG 115개 + SVG/SWF 벡터 + 스프라이트시트로 구성됩니다.

| 카테고리 | 개수 | 내용 |
|----------|------:|------|
| **Objects** | 42 | 표적 — 과녁(target), 오리(duck), 나무·구름 등 장식, 병/캔 표적 |
| **Stall** | 16 | 사격장 부스(나무 가판대), 가림막, 커튼, 배경 패널 |
| **HUD** | 48 | 크로스헤어(4색 × 대소), 총알 아이콘, 숫자/텍스트(SCORE, GAME OVER, READY 등) |
| **Spritesheets** | 3 | objects/stall/hud 시트 + XML |

> 크로스헤어 **4색 × 대/소**, 점수용 **비트맵 숫자/문구**까지 포함 → 조준·점수 UI를 추가 에셋 없이 완성.

---

## 2. 언제 사용하면 되는가?

- **사격장 / 표적 맞히기** 미니게임 (오리 쏘기, 과녁 사격)
- **조준 + 클릭/탭 슈팅** 캐주얼 게임
- 타이머·점수 기반 아케이드 (READY/GO/TIME UP/GAME OVER 문구 포함)
- 마우스 조준 크로스헤어가 필요한 모든 프로토타입

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **파일명이 의미 기반**(`crosshair_red_large`, `icon_duck`, `text_score`) → 식별·로딩 쉬움.
- **비트맵 텍스트/숫자 포함**(`text_0`, GAME OVER 등) → 폰트 없이 점수·상태 표시 가능.
- **크로스헤어 풀세트** — 색·크기 변형으로 다른 게임에서도 재사용 가치 높음.
- **Stall(부스) 배경** — 사격장 분위기 연출용 전경 오브젝트.
- **플랫 벡터 스타일** — 픽셀아트 아님.

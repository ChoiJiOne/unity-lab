# Kenney Minimap Pack 에셋 가이드

미니맵용 픽셀 타일/아이콘 팩입니다.
지형·경로·마커(시작/목표/적/아이템)를 미니맵에 표시할 때 사용합니다.

- **경로:** `Assets/Arts/kenney_minimap-pack/`
- **제작자:** Kenney (www.kenney.nl) · Minimap Pack v1.0 (2024-11-05)
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

미니맵/맵 오버뷰용 **소형 픽셀 타일·마커 세트**입니다.
25종 타일 × 6스타일 = PNG 150개 + 스타일별 타일맵 시트로 구성됩니다.

| 구분 | 개수 | 내용 |
|------|------:|------|
| **Tiles / Style A~F** | 각 25 | 미니맵 타일 — 지형/벽/경로 + 마커(시작·목표·집·적·아이템·계단·X표시 등) |
| **Tilemap / Style A~F** | 각 2 | 스타일별 타일맵 시트 + packed |

> 6가지 **색/톤 스타일(A~F)** — 흑백, 블루, 레드 등. 미니맵 디자인에 맞춰 선택. `Tilesheet.txt` 매핑 포함.

---

## 2. 언제 사용하면 되는가?

- **미니맵 / 맵 오버뷰 HUD** — 던전·레벨 구조 축약 표시
- **마커 표시** — 시작 지점, 목표, 적 위치, 아이템, 보스
- 로그라이크·탐험 게임의 발견된 영역 표시
- 레이더/전술 화면의 심볼

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **6스타일(A~F)** — 톤/색이 달라 UI 테마에 맞춰 선택.
- **소형 픽셀 타일** — 임포트 시 `Filter Mode: Point`, `Compression: None` 권장.
- **파일명이 `tile_0000` 식 인덱스** — `Tilesheet.txt` / `Preview.png`로 매칭.
- **마커 타일 포함**(시작/목표/적/아이템/계단) — 기능적 심볼로 즉시 사용.
- **packed 타일맵** 제공 — Tilemap 슬라이스 효율적.
- micro-roguelike 등 본맵 타일과 **짝**으로 쓰면 좋음.

---

## 형태 선택 가이드

- **Unity Tilemap 미니맵:** `Tilemap/Style X/tilemap_packed.png` → Slice
- **개별 마커/타일:** `Tiles/Style X/` 낱개
- **스타일:** A~F 중 UI 톤에 맞게
- **참고:** `Preview.png`(스타일 비교), `Tilesheet.txt`(매핑), `Sample.png`(배치 예시)

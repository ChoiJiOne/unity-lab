# Kenney Map Pack 에셋 가이드

탑다운/2D 맵 제작용 타일 에셋 팩입니다.
다양한 지형(잔디·모래·눈·물)과 경로·장식 타일로 레벨 맵을 빠르게 구성할 수 있습니다.

- **경로:** `Assets/Arts/kenney_map-pack/`
- **제작자:** Kenney (www.kenney.nl) · Map Pack
- **라이선스:** **CC0 (Public Domain)** — 개인·교육·상업 사용 전부 자유, 크레딧 불필요(권장)

---

## 1. 어떤 종류의 에셋인가?

탑다운 맵용 **지형 타일 세트**입니다. 플랫/벡터 스타일.
PNG 188개(낱개 타일) + 타일시트 + 스프라이트시트 + SVG/SWF 벡터로 구성됩니다.

| 카테고리 | 개수 | 내용 |
|----------|------:|------|
| **PNG (mapTile_001~)** | 188 | 지형 타일 — 잔디/모래/눈/물/돌, 가장자리·코너 타일, 경로(path), 나무·선인장·버섯 등 장식, 아이템 칸 |
| **Tilesheet** | 2 | 타일맵 정렬 시트 (격자 정렬, 타일맵 임포트용) |
| **Spritesheet** | 1 | 스프라이트시트 + XML 매핑 |
| **Vector** | SVG/SWF | 무한 확대용 원본 벡터 |

> 지형마다 **모서리·코너 변형 타일**이 포함되어 자연스러운 경계 연결 가능.

---

## 2. 언제 사용하면 되는가?

- **탑다운 RPG / 어드벤처 맵** 제작
- **타일 기반 레벨 디자인** (Unity Tilemap + Tile Palette)
- 보드게임풍 격자 맵, 경로 기반 퍼즐
- 여러 바이옴(잔디/사막/설원/수상) 맵 프로토타입

---

## 3. 특이 사항

- **CC0 라이선스** — 상업적 사용 포함 자유.
- **파일명이 `mapTile_001` 식 인덱스** — 어떤 타일인지는 `Preview.png` / `Tilesheet`로 눈으로 매칭 필요.
- **Tilesheet 제공** — Unity Sprite Editor에서 격자 Slice 후 Tile Palette로 등록하기 편함.
- **지형 경계/코너 타일 포함** — 바이옴 간 전환을 매끄럽게 표현 가능.
- **플랫 벡터 스타일** — 픽셀아트 아님. 일반 필터(Bilinear) 권장.

---

## 형태 선택 가이드

- **Unity Tilemap 레벨 제작:** `Tilesheet/` → Slice → Tile Palette
- **개별 배치:** `PNG/` 낱개 타일
- **고해상도/벡터:** `Vector/` (SVG)
- **참고:** `Preview.png`(전체), `Sample.png`(배치 예시)

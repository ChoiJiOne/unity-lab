# CLAUDE.md

## 샘플 에셋 작업 지침

샘플 에셋(목록 조회, 특징 확인, 샘플 선택, 사용법 등) 관련 작업을 할 때는
먼저 아래 `ai-docs/` 문서를 참조한다. 각 문서에는 에셋의 종류·사용 시점·특이사항이 정리되어 있다.

| 에셋 | 문서 | 위치 |
|------|------|------|
| Spine 예제 스켈레톤 | [`ai-docs/spine-sample-assets.md`](ai-docs/spine-sample-assets.md) | `Assets/Packages/Spine/Spine Examples/Spine Skeletons/` |
| Kenney Input Prompts (입력 아이콘) | [`ai-docs/arts-input-prompts.md`](ai-docs/arts-input-prompts.md) | `Assets/Arts/kenney_input-prompts_1.5/` |
| Kenney Desert Shooter Pack (탑다운 슈터) | [`ai-docs/arts-desert-shooter-pack.md`](ai-docs/arts-desert-shooter-pack.md) | `Assets/Arts/kenney_desert-shooter-pack_1.0/` |
| Kenney Boardgame Pack (보드/카드 게임) | [`ai-docs/arts-boardgame-pack.md`](ai-docs/arts-boardgame-pack.md) | `Assets/Arts/kenney_boardgame-pack/` |

## 새 에셋 추가 시

새로운 샘플 에셋을 추가하면 다음을 수행한다.
1. 에셋의 프리뷰 이미지와 `License.txt`를 확인한다.
2. 위와 동일한 스타일(종류 / 사용 시점 / 특이사항)로 `ai-docs/`에 가이드 문서를 작성한다.
3. 이 표에 항목을 추가해 인덱스를 갱신한다.

- 샘플 아트 에셋: `Assets/Arts/`
- Spine 예제: `Assets/Packages/Spine/Spine Examples/`
